# Mobile Pulsa BCA Codeigniter 3

Penggunaan API Sandbox BCA untuk cek saldo dan mutasi rekening [disini](https://developer.bca.co.id/) dan API Sandbox Mobile Pulsa untuk cek saldo dan pricelist pulsa operator [disini](https://developer.mobilepulsa.net/).
[Lihat web](https://bcamobilepulsa.herokuapp.com/).
## Alat:
-  `menggunakan template admin bootsrap premium Hound`
-  `postman`
-  `library PHP guzzle (HTTP Request)`
## git clone:
-  `download project`
-  `gunakan laragon dan pindahkan di folder www atau xampp di htdocs`
-  `ganti base_url d config.php sesuai nama folder project`
-  `done`
